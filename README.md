# TextEditor


 Text Editor to edit plain text files.
 

# Description:


Text Editor to edit Document, Searching.


Developed in Eclipse and Intellij idea.


Java JDK 15


# Applications: 


Text Editor can:

 1.Open File from your directories
 
 
 2.Create new file.
 
 
 3.Edit text.
 
 
 4.Predict next word.
 
 
 5.Search Data.
 
 
 6.Perform Undo/Redo Functionality.
